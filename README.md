# GestureX ✨
### AI-Powered Touchless Interaction System Using Hand Gestures

GestureX is a real-time computer vision application that allows users to control their computer without touching a mouse or keyboard. Using hand gesture recognition, users can perform various system operations through simple hand movements, creating a seamless and interactive experience.

🏆 **1st Prize Winner – AVISHKAR 2026 National-Level Hackathon**

---

## 📌 Overview

GestureX leverages Computer Vision techniques to detect and track hand movements in real time. The system translates specific hand gestures into computer commands, enabling touchless interaction with the operating system.

The project aims to enhance accessibility, improve user interaction, and provide a futuristic approach to human-computer communication.

---

## 🚀 Features

- Real-time hand tracking
- Touchless mouse cursor control
- Left click functionality
- Right click functionality
- Smooth cursor movement using interpolation
- Motion smoothing for accurate tracking
- Fast and responsive performance
- User-friendly interface

## ⚙️ How It Works

1. Webcam captures live video feed.
2. Hand landmarks are detected using Computer Vision.
3. Finger positions are analyzed.
4. Specific gestures are mapped to actions.
5. Corresponding mouse actions are executed.

Example:
MOUSE MODE
| Gesture | Action |
|----------|----------|
| Index Finger Up | Move Cursor |
| Index + Middle Finger | Right Click |
| Index + Thumb finger | Left Click |
|Open palm    | Change mode to presentation |
PRESENTATION MODE
| Index finger + Middle finger | Next page |
| Index finger + Pinky finger  | Previous page |
